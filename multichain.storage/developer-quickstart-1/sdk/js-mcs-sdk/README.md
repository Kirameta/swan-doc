# js-mcs-sdk

js-mcs-sdk 是一个用于 MCS 服务的 javascript 软件开发工具包。 它为从 Web 浏览器或 Node.js 使用 MCS API 提供了一个方便的界面。

SDK 具有以下功能：

* POST 上传文件到 FilSwan IPFS 网关
* POST 支付给 Swan Filecoin 存储网关
* 铸造为 NFT
* 获取上传的文件列表
* 按 CID 获取文件
* 从 Filecoin 获取存储状态
