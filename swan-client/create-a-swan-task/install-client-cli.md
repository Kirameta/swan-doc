# Install Client CLI

## Prerequisite

* Lotus node
* go 1.15+
* python 3.7+
* pip3

## Installation

#### Ubuntu/Debian

```
sudo apt-get update
sudo apt-get upgrade -y

# Install Git
sudo apt install git -y

# Checkout the source and install
git clone https://github.com/filswan/swan-client.git
cd swan-client

sh install.sh

. ./activate 
```
