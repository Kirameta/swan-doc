# Run in autobid mode

Autobid is a simplified process for swan provider who can fast join the auction process.

![Task Distribution to multiple Swan Provider](<../../.gitbook/assets/image (25).png>)

```
vi ~/.swan/provider/config/config.toml
```

set `  bid_mode = 1  `
