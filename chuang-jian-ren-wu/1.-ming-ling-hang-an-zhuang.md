# 1.命令行安装

## 代码库地址

[https://github.com/filswan/swan-client](https://github.com/filswan/swan-client)

## 必备安装条件

* Lotus node
* go 1.15+
* python 3.7+
* pip3

## 命令行安装

#### Ubuntu/Debian

```
sudo apt-get update
sudo apt-get upgrade -y

# Install Git
sudo apt install git -y

# Checkout the source and install
git clone https://github.com/filswan/swan-client.git
cd swan-client

sh install.sh

. ./activate 
```
